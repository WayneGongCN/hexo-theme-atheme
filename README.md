## OS theme

简洁小巧的复古主题。

Simple and light old style theme.

### Use

主题安装
```shell
cd /hexo-dir

# install theme
git clone https://github.com/zowiegong/hexo-theme-os.git /themes/os

# install depend
npm install hexo-renderer-github hexo-renderer-jade hexo-filter-asset-inline --save
```


修改 Hexo 配置文件 theme 字段
```yml
theme: os
```
